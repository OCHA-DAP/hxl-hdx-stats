Collect HXL stats from HDX
==========================

## Requirements

- Python3
- make

## Usage

Generate reports:

    $ make

Regenerate reports:

    $ make clean reports

Update Python packages:

    $ make real-clean

Push commits to GitHub:

    $ make sync

## Web links

- http://hxlstandard.org
- https://data.humdata.org
